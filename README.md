# Love STEM

## Set Up

- Have npm installed in your machine.
- Set up the tailwind;

```sh
 curl -L -o tailwindcss https://github.com/tailwindlabs/tailwindcss/releases/latest/download/tailwindcss-linux-x64
```

make it executable and execute it;

```sh
./tailwindcss -i ./src/index.css -o ./src/output.css -w
```

Next npm install and run the application.
